    Alternative method.

  1. Save this as a bookmark! (this is a bookmarklet, a piece of javascript code that is ran in the url)
  2. Go to HumbleBundle's Key's page. Login if you haven't already.
  3. Select your search options on HumbleBundle's Keys page, such as:
      Most Recent or Alphabetical
      redeemed or non redeemed keys
  4. Now open the bookmark you saved in step 1.
      If using a mobile device, open the bookmark by typing the name of the bookmark in the address bar and click on it when available.
  5. A red button "list to clipboard" should now be displayed, click the button to copy the list to your clipboard.
      Please note, you will have to open the bookmark everytime you refresh or revisit the page.
